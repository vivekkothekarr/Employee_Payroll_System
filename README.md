# 💼 Employee Payroll System

## 📌 Overview
A simple web-based payroll system to manage employees, salaries, deductions, bonuses, and payslip generation.  

## ⚙️ Technologies
- Python (Flask)
- SQLite Database
- HTML & CSS
- Oracle SQL (for schema design)

## ✨ Features
- Add Employees  
- Add Payroll Records  
- Calculate Net Salary  
- Generate & Print Payslip  
- View Full Database
